Guide d'utilisation

1. Comment recevoir un email ?

Votre boîte de réception dans l'onglet "Mes Données" écoute en temps réel. Dès qu'un mail arrive sur nos serveurs pour votre ID, il s'affiche.

2. Est-ce anonyme ?

Oui. Si vous utilisez le mode invité, aucune trace n'est conservée après déconnexion.

3. Comment simuler un test ?

Cliquez sur le bouton doré "Simuler Réception" dans votre profil pour vérifier que la connexion fonctionne.
